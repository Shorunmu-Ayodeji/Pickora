# Pickora

**Fair Picks. Real Winners.**

A clean, minimal raffle web app designed for X giveaways and community raffles.

## Features

- Secure random winner selection using `crypto.getRandomValues`
- Clean black & white minimal design
- Mobile-friendly and responsive
- Copy formatted tweet functionality
- Animated winner reveal

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Paste a list of names or X handles (one per line)
2. Choose how many winners to pick
3. Click "Pick Winners"
4. Copy the formatted tweet to share results

## Technology

- React 18
- Vite
- CSS3 (no external styling libraries)
